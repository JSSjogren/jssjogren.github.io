Project Gaze - Repository

Project Gaze - Prototypes Built

Explanation: This video just shows a brief update of everything I have accomplished so far with Project Gaze. There is certainly a lot left to learn and to create, but it has been an awesome learning experience so far!

What have I accomplished so far? Since this is largely a learning experience for me, I have spent a lot of time working on the fundamentals. I have setup basic local co-op, implemented the Gaze and its functionalities, created some basic UMG, and attached the scoreboard to the statistics for each player. Each player will also spawn with a unique color to help differentiate on the scoreboard.

What's next? I believe the next step for Project Gaze will be cleaning up the look of the game. I would love to create, rig, and animate a new model for the character to help the game feel more unique. After that, I will be looking into implementing other fun gameplay features such as in-game perks, different game modes, and possibly different classes.

About Project Gaze

Project Gaze is an indie game group/project that I started fully working on this last month. The goal of the project is to create a quick, fun couch co-op game, which I hope to be as addicting as other couch co-op games like Gang Beasts.
