Directed Study - Escape Room

Explanation: This is a project I created during a directed study class under Dr. Maria Hybinette. This project was mostly meant to learn a bit more about Unreal Engine's blueprint system, as I already had some experience with its C++ API via Project Tall Grass. The features implemented were recommended by Dr. Hybinette as good learning points as they allowed me to learn a bit more about UMG, collisions, and physics in Unreal Engine.
